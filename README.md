# Online-Book-Store-SQL-Project
Online Book Store Management System built with SQL, including book inventory management, customer records, order processing, and sales analytics.

This project is a SQL-based Online Book Store database designed to manage and analyze bookstore operations efficiently. The database is built using three core datasets: Books, Customers, and Orders, which together simulate the day-to-day activities of an online bookstore.

The Books dataset contains information about book titles, authors, genres, publication years, prices, and stock availability. The Customers dataset stores customer details such as names, contact information, cities, and countries. The Orders dataset records purchase transactions, including order dates, quantities purchased, and total order amounts.

Using SQL, the project establishes relationships between these datasets to track customer purchases, monitor inventory levels, analyze sales performance, and generate meaningful business insights. Various SQL queries are used to identify top-selling books, evaluate customer purchasing patterns, calculate revenue, and support data-driven decision-making.


Key Features:

* Book inventory management and stock tracking
* Customer information management
* Order and sales transaction tracking
* Revenue and sales performance analysis
* Customer purchase behavior analysis
* Relational database design using SQL
* Business reporting and data-driven insights

Dataset Summary:

* 500 Book Records
* 500 Customer Records
* 500 Order Records

This project showcases how SQL can be used to organize, manage, and analyze data in a real-world e-commerce environment.
